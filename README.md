# Student-Management-System
Aplicación desarrollada en Java para administrar información de estudiantes dentro de una base de datos. Permite registrar, consultar y actualizar información académica de los estudiantes.  
Tecnologías utilizadas: 
- Java
- SQL  
Funcionalidades:
- Registro de estudiantes
- Consulta de información
- Actualización de datos
- Eliminación de registros.
